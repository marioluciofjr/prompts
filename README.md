## Introdução

Repositório com uma coleção de prompts prontos para uso em ferramentas de IA generativa como ChatGPT, Gemini, Claude, Grok e similares. Os prompts cobrem diferentes finalidades: criação de conteúdo para LinkedIn, resumo de artigos científicos, geração de imagens, negociação, metodologias ágeis, estratégias de SEO e muito mais. Cada arquivo é um template reutilizável com variáveis substituíveis.

## Estrutura do projeto

O repositório está organizado em dois níveis: arquivos de prompt na raiz e uma pasta dedicada a prompts de geração de imagens.

Na **raiz do repositório**, estão os prompts para uso em IAs de texto. Cada arquivo `.md` ou `.txt` representa um template independente com variáveis entre chaves `{}` ou tags XML para facilitar a personalização. Os prompts cobrem áreas como criação de posts para o LinkedIn (`post_linkedin.md`, `linha_editorial_linkedin.txt`), resumo de livros (`resumelivros.md`), análise de artigos científicos (`papers.md`), mapa de empatia (`mapa_de_empatia.md`), matriz de stakeholders (`stakeholders.md`), geração de personas (`personasIAs.txt`), topic clusters para SEO (`topic_clusters.md`), criação de músicas no Suno AI (`criar_musica.md`), verificação de qualidade de prompts (`verificador1.md`, `verificador2.md`), entre outros.

A pasta **`IAs_de_Imagem/`** reúne prompts voltados exclusivamente para ferramentas de geração de imagens. Os templates cobrem estilos variados como infográficos, figurinhas, fotografias de estúdio, personagens 3D, capas de LinkedIn, HQs e muito mais.

## Requisitos

Antes de começar, certifique-se de ter instalado:
* [Git](https://git-scm.com/downloads) — para clonar o repositório localmente
* [VSCode](https://code.visualstudio.com/download) — editor recomendado para visualizar e editar os arquivos Markdown
* Conta em ao menos uma ferramenta de IA generativa — [ChatGPT](https://chat.openai.com), [Gemini](https://gemini.google.com), [Claude](https://claude.ai) ou similar
