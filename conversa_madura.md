# Conversa Madura
Este prompt serve para explicar para um LLM o que não gostou em uma saída e como gostaria que fosse. A inspiração para este prompt surgiu depois de ouvir o episódio ["Aprendemos mais quando somos punidos?"](https://www.b9.com.br/shows/naruhodo/naruhodo-281-aprendemos-mais-quando-somos-punidos/), do Naruhodo Podcast.

```
comportamento atual: [COLOQUE AQUI UM COMPORTAMENTO DA SAÍDA DO LLM QUE TE INCOMODOU]
comportamento esperado: [COLOQUE AQUI O COMPORTAMENTO DE SAÍDA QUE VOCÊ ESPERAVA] 

Por que você tem esse comportamento atual?
O que eu preciso fornecer de contexto ou regras para que o comportamento esperado seja executado de fato na tarefa?
```
